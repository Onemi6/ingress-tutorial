# Power Cube

Power Cube 是快速补充 XM 的道具。

![Power Cube](images/power_cube.png)

## 基本参数

| 属性 | 值 |
|-|-|
| 掉率 | Very Common |
| 占用空间 | 1 |
| 等级 | L1 - L8 |
| 可叠加 | 同等级 |
| 排序方式 | 无 |
| 操作 | Use/Drop/Recycle |

## 功能

点击 Use 即快速回复相应等级的 XM，是 Agent 在攻击和防御过程中恢复 XM 的主要手段。

## Hack 产出

Hack Portal 时能获得自身等级和 Portal 等级两者之中较低者的 Power Cube。
